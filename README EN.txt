2. Installation.

If you update F3Site to newer version, don't install it again but go to UPDATES folder.

If it's a new installation, upload files from UPLOAD directory to server. You can delete unwanted skins (except CONTINUOS) and languages. Chmod CFG and CACHE directory to 0777 and all files inside to 0766. Then go to INSTALL catalog in your web browser, e.g.:
- http://your.site.com/install/
Remember: SQL user should have all privileges. After the installation, delete INSTALL directory and change settings in admin panel.
